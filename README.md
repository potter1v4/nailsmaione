# 💅 Nails Maione - Agendamento Online

Este é um projeto de **Landing Page** profissional desenvolvida para a **Nails Maione**, com foco em conversão e facilidade de agendamento para clientes de estética e nail design.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptado para celulares, tablets e desktops.
- **Sistema de Agendamento:** Integração direta com o WhatsApp, gerando mensagens automáticas com data, hora e serviço escolhido.
- **Portfólio:** Galeria de imagens para exibição dos trabalhos realizados.
- **Seção Sobre:** Espaço para autoridade da profissional e conexão com o cliente.
- **UX/UI Clean:** Estética minimalista com tipografia elegante (`Playfair Display` e `Poppins`).

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização, layout flexbox/grid e efeitos de responsividade.
- **JavaScript:** Lógica de agendamento, formatação de data e integração com API do WhatsApp.
- **Google Fonts:** Tipografia selecionada para o nicho de luxo/beleza.
